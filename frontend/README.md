# Frontend: React & Vite Agent Interface

A modern, responsive chat interface built with React and Vite to interact with the LLM-powered agent backend.

## Tech Stack

- **Framework**: [React](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Logic**: Vanilla JavaScript/TypeScript

## Getting Started

### Installation

```bash
npm install
```

### Running in Development

```bash
npm run dev
```

### Building for Production

```bash
npm run build
```

## Features

- **Chat Interface**: Interactive message history with support for agent streaming.
- **Agent Integration**: Seamlessly communicates with the FastAPI backend for real-time AI interactions.
- **Fast Development**: Optimized Vite configuration for instant HMR.
- **Type Safety**: Fully typed with TypeScript.
